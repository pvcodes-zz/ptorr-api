PTORR-API

PTORR-API is a open sourced restful api for torrents. It is build on Python, Flask.

Repository | Support Server

METHODS

GET: https://ptorr-api.herokuapp.com/key=batman

It will return the JSON Object consisting of torr-id and torr-title

GET: https://ptorr-api.herokuapp.com/getorr=torr-id

It will return the torr-file to be downloaded.
